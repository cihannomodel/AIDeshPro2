## 🚀 Quick Start

### Option 1: One-Click Start (Recommended)

**Windows Users:**
```bash
# Double-click start-local.bat
```

**Mac/Linux Users:**
```bash
chmod +x start-local.sh
./start-local.sh
```

### Option 2: Manual Setup

```bash
cd server
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
